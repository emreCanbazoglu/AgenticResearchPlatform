## How to use in a new session

1. Read `../spec.md`, `../plan.md`, and `../tasks.md`.
2. Open this folder and pick the first phase with incomplete items.
3. Follow that phase doc in order: scope, deliverables, tests, and exit criteria.
4. Run validation commands from `../quickstart.md`.
5. Update `../tasks.md` and `../research.md` before ending the session.
